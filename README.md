##  Featured Projects

###  [LRC — C++ Frontend Compiler with AST Interpreter](https://github.com/moataz440/LRC-C-Compiler-with-AST-Interpreter-)
> An educational C++ compiler built entirely in Python, covering the full compilation pipeline.

- **Preprocessing → Lexing → LR(1) Parsing → Semantic Analysis → TAC → Pseudo-Assembly → HTE Object Code**
- Tree-walking AST interpreter for direct execution — no linker needed
- Full GUI built with CustomTkinter visualizing every compilation phase interactively
- Structured error reporting at each phase with precise line/column info and fix suggestions
- **Stack:** Python · LR(1) Parsing · CustomTkinter

---

###  [Car Dealership Platform](https://github.com/moataz440/Car-Dealership)
> A full-stack car marketplace with listings, auctions, bidding, and lead management.

- Full-stack app with a Next.js frontend and Node.js/Express REST API backend
- MySQL database with relational schema (listings, users, bids, comments, leads), managed via Prisma ORM
- JWT authentication with bcrypt password hashing
- Real-time auction system with bidding, comments, and listing status management
- **Stack:** Next.js · TypeScript · Tailwind CSS · Node.js · Express · MySQL · Prisma · REST API
